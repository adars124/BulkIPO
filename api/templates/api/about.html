Let's get connected!

Welcome to BulkIPO, which is a web application that is built on top of Django. It allows users to apply IPOs in bulk from multiple different accounts at once. It is built using the MeroShare api in the backend which is used from authentication to applying the shares. The client's information is stored in the db.sqlite3 database file stored in the SQLITE3 format which is locally saved on the users project file if cloned this repos.
